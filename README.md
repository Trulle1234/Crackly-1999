# Crackly-1999

A "port" of [the website for my font Crackly](https://trulle1234.github.io/Crackly/) made for _[Like it's 1999](https://99.hackclub.com/)_. It is made entirely (exept gifs) in Windows 98 using EditPlus for code and Intenet Explorer to veiw. For this one I heavily leaned into the crappy Geocites website look. To use it in Windows 98 just go here: [99.hackclub.com/emulator/index.html?share...](https://99.hackclub.com/emulator/index.html?share=https://github.com/Trulle1234/Crackly-1999/raw/refs/heads/main/like-its-1999-state_26-07-13_15-16.bin.gz)
